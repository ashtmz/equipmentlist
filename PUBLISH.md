# オンライン公開

このフォルダは静的サイトとしてそのまま公開できます。

## 最短手順

1. Netlify にログイン
2. `Sites` -> `Add new site` -> `Deploy manually`
3. このフォルダ `/Users/tmz/Set/Git/機材表` をそのままドラッグ&ドロップ
4. 公開URLが発行されたら完了

## このフォルダに入っている公開設定

- `netlify.toml`
  - 公開ディレクトリをルート (`.`) に設定
  - `index.html` は毎回更新確認
  - `app.js` / `styles.css` は長めキャッシュ
- `robots.txt`
  - 公開後に検索エンジンからアクセス可能

## 補足

- このアプリのデータはブラウザの `localStorage` に保存されます
- 別の端末や別ブラウザへは自動同期されません
- 複数人で共有するには、将来的にサーバー保存機能の追加が必要です
